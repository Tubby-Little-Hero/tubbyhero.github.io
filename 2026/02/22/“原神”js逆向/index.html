RSA 加密的核心是公钥（n, e） 和私钥（n, d）：

字段 含义（JS 逆向中常见叫法） 作用
n 模数（Modulus） RSA 的核心大整数，公钥 / 私钥都依赖它，通常是几百位的十进制大数
t（0-36） 大概率是n的分段拆分值 前端为了传输 / 处理方便，会把超大的n拆成多个小段存储（比如按位数 / 长度拆分）
s:0 状态位（成功标识） 表示参数生成 / 拆分成功
t:37 分段总数 说明n被拆成了 37 段（0-36）
